`ptmod-gui` is a graphical editor for polytaxis tags.

# Installation

## Linux

1. Install Python 3, pip, Qt, and `pyqt5` using your distribution's package manager.
2. Run
```
pip install https://github.com/Rendaw/ptmod-gui.git
```
